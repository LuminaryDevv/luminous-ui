✨ Luminous UI

A modern UI framework for Roblox executor scripts with 17 premium themes

🎨 Theme Gallery

📦 Installation

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/LuminaryDevv/luminous-ui/main/Main.lua"))()

🚀 Quick Example

local Window = Library:CreateWindow({
    Title = "My Script",
    Size = UDim2.fromOffset(570, 370),
    Theme = "Galaxy"
})

-- Change themes anytime
Window:SetTheme("Diamond")
Window:SetTheme("Gold")
Window:SetTheme("Galaxy")
