A modern, lightweight UI framework for Roblox executor scripts. Built for performance and cross-executor compatibility.

## 🌟 Features

- 🪟 **Full window system** – drag, resize, minimize, maximize, close
- 🎨 **17+ UI components** – buttons, toggles, sliders, dropdowns, keybinds, inputs, sections, paragraphs
- 🌗 **17 Premium Themes** – Diamond, Gold, Silver, Ruby, Emerald, DeepSea, Amethyst, Topaz, Violet, Sky, Camo, Galaxy, Rainbow, Wooden, Dark, Light, Void
- 🔔 **Notification system** – timed popups with animations
- 🌀 **Blur effects** – with graceful fallback
- ⚡ **Executor-friendly** – works on Solara, Wave, Delta, Swift, Xeno, Ronix, and more
