# Luminous-ui by LuminaryDevv
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Roblox](https://img.shields.io/badge/Roblox-Executor%20UI-red)](https://github.com/LuminaryDevv/luminous-ui)

A modern, lightweight UI framework for Roblox executor scripts. Built for performance and cross-executor compatibility.

## ✨ Features
- 🪟 **Full window system** – drag, resize, minimize, maximize, close
- 🎨 **8+ UI components** – buttons, toggles, sliders, dropdowns, keybinds, inputs, sections, paragraphs
- 🌗 **Theme system** – dark/light/camo/sapphire/diamond/galaxy/gold
- 🔔 **Notification system** – timed popups with animations
- 🌀 **Blur effects** – with graceful fallback
- ⚡ **Executor-friendly** – works on Solara, Wave, Delta, Swift, Xeno, Ronix, Bunni and more.
